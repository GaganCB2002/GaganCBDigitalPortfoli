# Gagan C B - Digital Portfolio
A high-performance, interactive personal portfolio website designed to showcase my skills, projects, and professional background as a Full Stack Java & Web Developer.

## 🚀 Features
- **Premium Dark Theme**: A sleek, modern user interface with a "Midnight Slate" color palette.
- **Modern Scroll Animations**: Lightweight, scroll-triggered animations powered by IntersectionObserver.
- **Interactive Elements**:
  - **Mouse Spotlight**: A dynamic glow effect that follows the cursor.
  - **Terminal Loader**: A unique "System Boot" loading sequence.
  - **Typing Text**: Animated hero text for a dynamic introduction.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Project Showcase**: Dedicated cards with hover effects to display academic and personal projects.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Premium Custom Styling), JavaScript (Vanilla ES6+), React, TypeScript
- **Libraries**: Three.js, React Three Fiber, GSAP, Framer Motion
- **Tools**: Vite, Node.js

## ⚡ How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/GaganCB2002/GaganCBDigitalPortfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd GaganCBDigitalPortfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open in Browser:** 
   Go to `http://localhost:5173`
